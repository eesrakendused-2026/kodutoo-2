# Typer Mäng

Tunnis loodud Typer Mängu täiustamise kodutöö

## Funktsionaalsus

1. Erinevad fondid on kasutusel, kaasaarvatud Google fontid
2. Võimalik on näha leaderboardsi vajutades "Leaderboards" nupule, ning võimalik on seda sulgeda uuesti "Leaderboards" nupule vajutades, või kui avatakse "Customization" menüü
3. Leaderboardi on kaunistatud
4. Minimalistlik ja stiile CSS kujundus
5. Kasutusel mitu erinevat pseudo-classi
6. Kasutusel on 5 erinevat heliklippi, mida on võimalik kuulda siis kui:alustatakse mängu, vajutatakse klahvile, mängitakse mängu, lõpetatakse mäng positiivse tulemusega,lõpetatakse mäng negatiivse tulemusega  
7. Lisatud on "customization" menüü, kus on võimalik valida 2 erineva, ning default tausta vahel, ning võimalik on seda sulgeda uuesti "Customization" nupule vajutades, või kui avatakse "Leaderboards" menüü
8. Leaderboardsidele on lisatud filtreerimis süsteem, filtreeri aja ja skoori vahel VÕI raskusastmete vahel
9. On võimalik valida 4 erineva raskusastme vahel: Easy, Normal, Hard või Death Sentence. Mängu ajavõttu süsteem kohandub raskustasemest
