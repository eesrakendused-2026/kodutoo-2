# kodutoo-2

Täiustasin trükkimismängu ja tegin kõik ära vajalikud punktid.

## Mida tegin

### 1. Google font
Kasutasin Google fonti `Roboto`.

### 2. Tulemused nupule vajutades
Lisasin nupu `Vaata tulemusi`, mis avab tulemused modalis, ja `x` nupu, mis paneb selle kinni.

### 3. Tulemuste parem kuvamine
Tulemused on nüüd eraldi plokkides ja igal real on nimi, aeg, kiirus ja tase.

### 4. Pilt vastavalt kiirusele
Lisasin 3 erinevat pilti:
- aeglane
- keskmine
- kiire
Pilt on suvaliselt valitud, et näidata funktsionaalsuse töötamist.

### 5. Mobiilivaade
Lisasin media query, et väiksel ekraanil oleks paigutus lihtsam kasutada.

### 6. CSS muudatused
Muutsin värve, suuruseid, ääri ja vahekauguseid.

### 7. Vähemalt 5 pseudo classi
Kasutasin:
- `:hover`
- `:active`
- `:focus`
- `:disabled`
- `:checked`
- `:nth-child(even)`
- `:first-child`

### 8. 4 heli
Lisasin heli:
- mängu alguses
- kirjutamise ajal
- mängu lõpus
- siis kui tulemus läheb edetabelisse

### 9. lisafeature'id
Lisasin:
- raskusastme valiku
- täpsuse näitamise
- järgmise sõna eelvaate
- uuesti alustamise nupu
